  Here's how to use api :   
**Gradle**:
```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
```gradle
	dependencies {
	        implementation 'com.github.GaMeIsNtOvEr:SQLAPI:Tag'
	}
```
**Maven**
```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
``` 
```xml
	<dependency>
	    <groupId>com.github.GaMeIsNtOvEr</groupId>
	    <artifactId>SQLApi</artifactId>
	    <version>Tag</version>
	</dependency
```
